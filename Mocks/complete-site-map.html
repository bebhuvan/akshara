Implementation Guide

Static Site Generator

Hugo for fast builds. Content in Markdown. Git-based workflow. Build times under 5 seconds for hundreds of books.

Typography

Libre Baskerville (reading) + Inter (UI). Google Fonts with variable weights. Screen-optimized for extended reading.

Deployment

Cloudflare Pages with automatic Git integration. Global CDN. R2 for EPUBs and large assets. Edge caching.

Search

Pagefind for client-side search. No server needed. Works offline. Indexes all content at build time.

Color Palette

Warm paper (#fffef9), ink (#1a1412), terracotta accent (#c85a3a). Smoke and ash grays. Theme variations for dark/sepia.

Performance

Minimal JavaScript. Critical CSS inlined. Progressive enhancement. Lazy loading. Service Worker for offline reading.

Reading Features

Focus mode, keyboard shortcuts (T/S/F/arrows), reading time estimates, progress tracking, position memory via LocalStorage.

Content Structure

books/[slug]/_index.md for metadata, chapters/ subdirectory. Front matter for year, author, language, themes.

Mobile First

Designed for mobile-majority traffic. Touch-friendly controls. Responsive typography. Optimized for 3G networks.
